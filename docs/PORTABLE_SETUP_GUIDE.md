# Руководство по установке портативной версии ALPHACAM API SDK

## Требования к системе

### Обязательные требования:
- Windows 10 или выше
- ALPHACAM 2024 (установленная версия)
- .NET Framework 4.8 или выше

### Рекомендуемые требования:
- 8 GB RAM или больше
- 1 GB свободного места на диске
- Процессор Intel Core i5 или выше

## Процесс установки

### 1. Подготовка
1. Скопируйте файл `ALPHACAM_API_SDK_Portable.zip` на целевой компьютер
2. Создайте директорию для SDK (например, `D:\ALPHACAM API`)
3. Распакуйте архив в созданную директорию

### 2. Структура портативной версии
```
D:\ALPHACAM API\
├── bin\           # Скомпилированные библиотеки
├── lib\           # Зависимости и библиотеки ALPHACAM
├── help\          # Файлы справки (*.chm)
├── docs\          # Документация SDK
├── examples\      # Примеры использования
├── src\           # Исходный код SDK
└── portable_config.json  # Конфигурация SDK
```

### 3. Проверка файлов
1. Убедитесь в наличии всех необходимых файлов справки в директории `help`
2. Проверьте наличие исходного кода в директории `src`
3. Проверьте примеры в директории `examples`
4. Убедитесь, что конфигурационный файл `portable_config.json` существует

### 4. Настройка окружения
1. Откройте `portable_config.json`
2. Проверьте и при необходимости обновите пути:
```json
{
    "sdk_version": "1.0.0",
    "alphacam_version": "2024",
    "paths": {
        "installation": "./lib",
        "help": "./help",
        "examples": "./examples",
        "docs": "./docs"
    }
}
```

### 5. Проверка работоспособности
1. Откройте пример из директории `examples`
2. Попробуйте скомпилировать и запустить пример
3. Проверьте доступ к файлам справки

## Использование SDK

### 1. Подключение к проекту
```csharp
// Инициализация SDK
var sdk = SDKManager.Instance;
sdk.Initialize();

// Получение интерфейсов
var core = sdk.GetCoreInterface();
var geometry = sdk.GetGeometryInterface();
var automation = sdk.GetAutomationInterface();
```

### 2. Работа со справкой
```csharp
// Открытие справки
sdk.OpenHelp("ACAMAPI.chm");

// Открытие индекса справки
sdk.OpenHelpIndex();
```

### 3. Использование примеров
1. Изучите примеры в директории `examples`
2. Используйте их как основу для своих проектов
3. Обращайтесь к документации при возникновении вопросов

## Решение проблем

### 1. Проблемы с путями
- Проверьте пути в `portable_config.json`
- Убедитесь, что все директории существуют
- Проверьте права доступа

### 2. Проблемы с библиотеками
- Убедитесь, что ALPHACAM установлен
- Проверьте версию .NET Framework
- Проверьте наличие всех необходимых DLL

### 3. Проблемы со справкой
- Проверьте наличие CHM файлов
- Убедитесь, что файлы не заблокированы системой
- Попробуйте переустановить файлы справки

## Обновление SDK

### 1. Сохранение данных
- Создайте резервную копию ваших проектов
- Сохраните пользовательские настройки
- Экспортируйте важные данные

### 2. Процесс обновления
1. Удалите старую версию SDK
2. Распакуйте новую версию
3. Восстановите пользовательские настройки
4. Проверьте работоспособность
